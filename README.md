# Grok Landing

Professional landing page showcasing Grok's frontend development capabilities.

Built entirely by Grok using Next.js, TypeScript, Three.js, Framer Motion, and shadcn/ui-inspired components.

## Features
- Modern dark space theme inspired by X and Grok
- Immersive Three.js scrolling animations
- Smooth Framer Motion interactions
- Professional UI components
- Fully static export ready for GitHub Pages

This site is configured for GitHub Pages deployment.
